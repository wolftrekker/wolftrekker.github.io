Inclusive Fieldwork Map — Holyrood Park

Mapbox GL JS • QGIS • 2024

Project Overview

  • Interactive Web Mapping & Accessibility Design: Developed a fully functional web-based trail accessibility tool using Mapbox GL JS, implementing colorblind-friendly palettes, natural-language info panels, and WAI accessibility standards to support neurodivergent students in fieldwork planning.
  • Geospatial Data Integration & Processing: Combined multiple open data sources (OpenStreetMap, OS Terrain 5 DTM) in QGIS and R, creating a rule-based classification framework that integrated slope analysis, surface attributes, and landcover data to assess trail difficulty across 949 segments.
  • Advanced Routing Algorithms: Built a custom accessibility-weighted pathfinding system using Dijkstra's algorithm (Graphlib + Turf.js), enabling users to generate personalized routes that avoid steep slopes, exposed heights, and difficult surfaces based on individual preferences.
  • User-Centered Interface Design: Designed an intuitive toolbox with hierarchical filtering, dynamic legend updates, responsive info panels with ground-truth photography, and an interactive tutorial system—balancing technical functionality with clear, non-specialist communication.
  • Cartographic Design & Visualization: Created a simplified basemap in Mapbox Studio with strategic use of contours, muted landcover styling, and traffic-light/Paul Tol color schemes to ensure visual clarity while maintaining environmental context for 950+ classified trail segments.
